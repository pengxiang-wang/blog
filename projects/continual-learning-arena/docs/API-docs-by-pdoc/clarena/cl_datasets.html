Continual Learning Datasets

This submodule provides the continual learning datasets that can be used in CLArena.

Please note that this is an API documantation. Please refer to the main documentation page for more information about the CL datasets and how to use and customize them:

  • Configure your CL dataset: https://pengxiang-wang.com/projects/continual-learning-arena/docs/configure-your-experiments/cl-dataset
  • Implement your CL dataset: https://pengxiang-wang.com/projects/continual-learning-arena/docs/implement-your-cl-modules/cl-dataset
  • A beginners' guide to continual learning (CL dataset): https://pengxiang-wang.com/posts/continual-learning-beginners-guide#CL-dataset

The datasets are implemented as subclasses of CLDataset classes, which are the base class for all continual learning datasets in CLArena.

  • CLDataset: The base class for continual learning datasets.
  • CLPermutedDataset: The base class for permuted continual learning datasets. A child class of CLDataset.

513class Permute:
514    """Permutation operation to image. Used to construct permuted CL dataset.
515
516    Used as a PyTorch Dataset Transform.
517    """
518
519    def __init__(
520        self,
521        img_size: torch.Size,
522        mode: str = "first_channel_only",
523        seed: int | None = None,
524    ):
525        """Initialise the Permute transform object. The permutation order is constructed in the initialisation to save runtime.
526
527        **Args:**
528        - **img_size** (`torch.Size`): the size of the image to be permuted.
529        - **mode** (`str`): the mode of permutation, shouble be one of the following:
530            - 'all': permute all pixels.
531            - 'by_channel': permute channel by channel separately. All channels are applied the same permutation order.
532            - 'first_channel_only': permute only the first channel.
533        - **seed** (`int` or `None`): seed for permutation operation. If None, the permutation will use a default seed from PyTorch generator.
534        """
535        self.mode = mode
536        """Store the mode of permutation."""
537
538        # get generator for permutation
539        torch_generator = torch.Generator()
540        if seed:
541            torch_generator.manual_seed(seed)
542
543        # calculate the number of pixels from the image size
544        if self.mode == "all":
545            num_pixels = img_size[0] * img_size[1] * img_size[2]
546        elif self.mode == "by_channel" or "first_channel_only":
547            num_pixels = img_size[1] * img_size[2]
548
549        self.permute: torch.Tensor = torch.randperm(
550            num_pixels, generator=torch_generator
551        )
552        """The permutation order, a `Tensor` permuted from [1,2, ..., `num_pixels`] with the given seed. It is the core element of permutation operation."""
553
554    def __call__(self, img: torch.Tensor) -> torch.Tensor:
555        """The permutation operation to image is defined as a callable object like a PyTorch Transform.
556
557        **Args:**
558        - **img** (`Tensor`): image to be permuted. Must match the size of `img_size` in the initialisation.
559
560        **Returns:**
561        - The permuted image (`Tensor`).
562        """
563
564        if self.mode == "all":
565
566            img_flat = img.view(
567                -1
568            )  # flatten the whole image to 1d so that it can be applied 1d permuted order
569            img_flat_permuted = img_flat[self.permute]  # conduct permutation operation
570            img_permuted = img_flat_permuted.view(
571                img.size()
572            )  # return to the original image shape
573            return img_permuted
574
575        if self.mode == "by_channel":
576
577            permuted_channels = []
578            for i in range(img.size(0)):
579                # act on every channel
580                channel_flat = img[i].view(
581                    -1
582                )  # flatten the channel to 1d so that it can be applied 1d permuted order
583                channel_flat_permuted = channel_flat[
584                    self.permute
585                ]  # conduct permutation operation
586                channel_permuted = channel_flat_permuted.view(
587                    img[0].size()
588                )  # return to the original channel shape
589                permuted_channels.append(channel_permuted)
590            img_permuted = torch.stack(
591                permuted_channels
592            )  # stack the permuted channels to restore the image
593            return img_permuted
594
595        if self.mode == "first_channel_only":
596
597            first_channel_flat = img[0].view(
598                -1
599            )  # flatten the first channel to 1d so that it can be applied 1d permuted order
600            first_channel_flat_permuted = first_channel_flat[
601                self.permute
602            ]  # conduct permutation operation
603            first_channel_permuted = first_channel_flat_permuted.view(
604                img[0].size()
605            )  # return to the original channel shape
606
607            img_permuted = img.clone()
608            img_permuted[0] = first_channel_permuted
609
610            return img_permuted
